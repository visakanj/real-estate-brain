## How to Add Links

Copy the template below and add your entry to the appropriate section:

```markdown
### [Title of Resource]
| Field | Value |
|-------|-------|
| **URL** | [link] |
| **Type** | Article / Blog Post / Video / Podcast / Website / Tool |
| **Topic** | [e.g., Financing, Market Analysis, Legal, Tax Strategy] |
| **Added By** | [Your name] |
| **Date Added** | [YYYY-MM-DD] |
| **Priority** | High / Medium / Low |
| **Status** | Unread / In Progress / Read |

**Why it's relevant:**
[Brief note on why this is worth reading]

---
```
